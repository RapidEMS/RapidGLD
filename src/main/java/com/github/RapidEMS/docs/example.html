An Example GLD Session

In this section, you can follow a step-by-step detailed tutorial for GLD, from map creation all the way to evaluation. We will check out all the functions of GLD, and give a guideline to working realistically with infrastructures and traffic.

The following maps, named Simple and Complex, are used here and in the Algorithms section as example.
In the Algorithms section examples statistics are given for all algorithms so that you can tweak with these maps and learn how the TLC's and DP's work out in practice without running hours of simulations yourself.
