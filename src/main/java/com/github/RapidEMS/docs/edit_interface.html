Editor Interface

The Menus in the Editor give the following options, with their corresponding shortcut keys:



FILE
  • New
    		• for creating a map file from scratch [Ctrl + N]
  • Open...
    		• for opening a map file [Ctrl + O]
  • Save
    		• for saving a map file [Ctrl + S]
  • Save as...
    		• for saving a map under a different name than its last save
  • Properties
    Title
    Author
    Comments
    		•  for editing additional info on a map
  • Close
    		• for closing the Editor window [Ctrl + W]
  • Quit
    		• for quitting GLD [Ctrl + Q]


    EDIT
  • Delete
    		• remove selected portion of infrastructure [Ctrl + Delete]
  • Select All
    		• select all Roads and Nodes [Ctrl + A]
  • Deselect
    		• reset any selection [Ctrl + D]


    OPTIONS
  • Toggle Grid
    		• switch grid overlay for reference on
  • Change Size
    		• change the size of the map
  • Validate
    		• check all couplings between Roads and Nodes
  • Settings
    Config dialog always on top
    Road users cross nodes
    Sign controller switches
    traffic lights safely
    Statistics export separator character
    		• general configuration


    HELP
  • Help
    		• aid for the common user [Ctrl + H]
  • Specifications
    		• precise guide to the GLD code
  • License
    		• the open-source limitations for GLD
  • Website
    		• opens the GLD www adress in a web browser
  • About
    		• a little note about GLD

    The Editor's Toolbar contains the following:

    • The New button
    • The Load button
    • The Save button
    • The Zoom Box - displays current zoom factor (initialised at 100%) and allows adjustment to values:
      • 25%
      • 50%
      • 75%
      • 100%
      • 150%
      • 200%
      • 250%
    • The Center button: click to center the map
    • Zoom - the Tool for manually zooming to the above mentioned zoom factors
    • Select - the Tool for selecting nodes or roads
    • Node - the Tool for creating nodes
    • Road - the Tool for creating roads
    • Drive lanes tool - for adjusting the number of drive lanes of an existing road
    • The Help button: quick access to the documentation

    • The Tool Specific Box appears when the node tool is selected with -
      • Edge node - make next/selected node an edge node
      • Traffic Lights - give next/selected node traffic lights
      • No Signs - remove from next/selected node traffic lights
      • Net-Tunnel - for "transferring road user agents to a GLD simulation on another machine
