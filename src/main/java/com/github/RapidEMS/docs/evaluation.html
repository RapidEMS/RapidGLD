Evaluation Tools

The data collected from running a simulation on a map can be displayed and analyzed in a number of ways:

  • In-view statistics can be toggled on, which will color junctions to indicate the average waiting time there. The darker the color, the longer road users have to wait there. Beside this, the following info will be displayed in the top-left portion of the simulator screen:
    • largest waiting queue, with the ID of the junction
    • largest average junction waiting time, with the cycle count
    • total number of road users arrived, with count

  • The Statistics window can be opened, containing precise data on the current simulated map:
    • name of map, and author
    • cycle when the Statistics window was opened
    • list of edge nodes with: ID, arrived road users, current queue length, Total Road Users Arrived, and average trip waiting time
    • list of junctions with: ID and average junction waiting time (best to be over viewed by toggling in-view statistics, see above
    • totals of: number of road users arrived, current queue length, Total Road Users Arrived, average waiting time and average junction waiting time
    Alternatively, bar or pie charts can be opted for, instead of the standard table. The window can be refreshed to the current cycle of the simulation.

  • A tracking window to track one of the following:
    • total waiting queue length
    • Total Road Users Arrived
    • average trip waiting time
    • average junction waiting time
    Multiple windows can be opened, each automatically showing a chart that sets the selected info (on the Y-axis, updated every cycle) to the cycle time (X-axis, from the cycle the window was opened to a certain future cycle, updated when said cycle is passed)
