Building on Green Light District

GLD is only a small project, and the true power of traffic simulation and its analysis may only be unlocked in future versions, programmed by others.

For this reason GLD is open source, and extensive documentation is provided along the Java code we strove to make as clear as possible.

Expansions of use might be:

  • map creation by roads alone, without explicitly placing nodes
  • a map-recognizer so that (scanned) maps can be converted to GLD infrastructures
  • set timers and routes, so that bus- lines can be better simulated
  • fully customizable statistics, that display any chosen function mapped to any chosen subset of the infrastructure (interesting in connection with the former upgrades)
  • bots that supervise real time traffic, specifically frequently occurring jams, simulate the environment, and propose infrastructure upgrades
  • self-driving cars that co-operate on an absolute scale, with all possible knowledge

Here follows a list of other projects in the same area of research that might be of use:

  • Optimal traffic control research.
  • A simple traffic simulation.
  • A data visualization project in modern traffic information systems
  • Advanced traffic control simulations for roadways.
  • Computerized traffic control systems of Ontario, Canada.
