From a Simulation to an Evaluation

After creating a map and running a simulation, it is time to crunch numbers: select Simulation - Statistics from the menu.

The statistics can be viewed as tables for precision, or switched to graphics for quick overview.

Compare different map layouts based on one idea, like the Paris roundabout-center, or the New York island-mainland.

Experiment with different algorithms for road users and traffic lights, and weigh the results.
